# utila-island-cleanups
Utila Island Cleanups is a static website dedicated to promoting weekly beach cleanups in Utila, Honduras. The site shares the mission, impact data, and ways to get involved in protecting coastal ecosystems and wildlife through community-driven environmental action.

# Utila Island Cleanups

Official website for Utila Island Cleanups.

🌱 Creating awareness one cleanup at a time.

## Mission
We are committed to protecting the beaches of Utila, Honduras through weekly cleanups and responsible waste management.

## Features
- Bilingual content (English / Spanish)
- Weekly cleanup information
- Impact tracking (connected to data source)
- Volunteer and partnership opportunities
- Photo gallery

## Deployment
This site is designed to be hosted using GitHub Pages.
